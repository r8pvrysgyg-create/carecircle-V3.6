# CareCircle v3.6 — Directory + Appointment Auto-Fill

This release adds a general-purpose Directory for important contacts and phone numbers.

## New Features

- General Directory for doctors, pharmacies, Maple Ridge, Para Transit, family members, insurance contacts, hospitals, and any other important phone number.
- Directory fields include name, category, phone, alternate phone, address, website, default appointment type, default location, and notes.
- Appointment form now includes a Directory contact dropdown.
- Choosing a Directory contact auto-fills appointment type, visit details, location, and useful contact notes.
- Directory contacts appear in global search.
- Admins can delete Directory contacts. Approved family members can add and view contacts.

## Firebase Rules

Update Firestore rules with the included `firestore.rules` file.

## Deployment

1. Upload all files to your GitHub repository root.
2. Commit once.
3. Wait for GitHub Pages to deploy.
4. Hard refresh the site with Ctrl + Shift + R.
5. Test with your admin and family accounts.
